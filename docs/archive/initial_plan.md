# fx-auto-trading 初期仕様書

## プロジェクト概要

USD/JPYデイトレードに特化したFX自動売買システム。機械学習による価格方向予測の精度を最高峰に引き上げることを最優先目標とする。

## 動機

- 前バージョン（v1）は機能を広げすぎて複雑化し、予測精度が不十分だった
- v2ではスコープを絞り、予測モデルの精度に全集中する
- デイトレード（1時間足）に特化し、ノイズの少ないシグナルを目指す

## コア機能

1. **データ収集**: GMO FX Public API から USD/JPY 1H OHLCデータを取得・蓄積
2. **特徴量エンジニアリング**: テクニカル指標、時間帯特徴量、モメンタム指標等を生成
3. **予測モデル**: LightGBMベースの方向予測（上昇/下落）、ウォークフォワード検証
4. **評価**: バックテストによる実用性検証、各種メトリクス（精度、シャープレシオ等）

## アーキテクチャ

```
GMO FX API → データ収集 → 特徴量生成 → モデル訓練/予測 → 評価
```

シンプルなパイプライン構成。各ステップを独立モジュールとして実装し、個別にテスト可能にする。

## 技術要件

- Python 3.12+
- パッケージ管理: uv
- ML: LightGBM（メイン）、scikit-learn（前処理・評価）
- データ: pandas, httpx
- ストレージ: SQLite
- テスト: pytest

## 配布方法

CLI ツールとしてローカル実行。将来的にcron/systemdで定期実行。
