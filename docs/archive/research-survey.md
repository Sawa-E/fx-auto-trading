## 横断分析1: 「何を予測するか」

| 資料 | 主張 | 根拠 |
|------|------|------|
| 資料3 | 定常的な系列を予測せよ | 共和分スプレッドの予測は価格予測より高精度 |
| 資料4 | 勝率を最大化せよ | 変数重要度で勝率が圧倒的1位 |
| 資料2 | 統計的優位性のある取引を繰り返せ | 長期利益の源泉 |
| 資料1 | 単純な実装では損失が出る | 損切りロジックと特徴量が鍵 |
| 資料5 | 意味のないデータを入れるな。変化の局面で反応できるかが重要 | XGBoost/LightGBMが為替予測で最良。3値分類も有効 |

**統合結論**:
- 予測対象は**価格の方向（上/下）の2値分類**が基本。**3値分類（横ばい含む）も検討**（資料5）
- ただし**ノイズを除外する閾値**を設け、一定以上の値動きのみをラベル化
- モデルの**予測確率で取引を絞る**ことで実効勝率を引き上げる
- テクニカル指標の**均衡からの乖離度**を特徴量に組み込み、平均回帰性を活用
- **計量分析で学習データを事前選別**してからMLに投入（資料5）

## 横断分析2: 「どう評価するか」

| 資料 | 主張 | 根拠 |
|------|------|------|
| 資料3 | 十分な期間のアウトオブサンプルテスト | 2007-2017年の10年間で検証 |
| 資料2 | バックテストを過信するな | 過学習が最大リスク |
| 資料4 | 勝率×RR比で評価せよ | この組み合わせがパフォーマンスを決定 |

**統合結論**:
- **ウォークフォワード検証**が必須（時系列でのデータリーク防止）
- 評価指標は精度だけでなく**勝率 × リスクリワード比 × シャープレシオ**の組み合わせ
- 少額**フォワードテスト**（ペーパートレード）フェーズを設ける

## 横断分析3: 「過学習をどう防ぐか」

| 資料 | 過学習対策 |
|------|----------|
| 資料3 | 人工データ事前学習（ノイズ除去） |
| 資料2 | 少額フォワードテスト |
| 資料4 | ランダムフォレストの変数重要度で特徴量選択 |
| 資料1 | シンプルなモデルから始める |
| 資料5 | 複数モデル比較 → 引っ掛かりの良いモデルの上位版で精度向上 |

**統合結論**:
- 特徴量は**変数重要度ベースで選択**（不要な特徴量は除外）
- ウォークフォワード検証でアウトオブサンプル性能を確認
- LightGBMの**正則化パラメータ**を適切に設定
- Optunaによるハイパーパラメータチューニングはウォークフォワードの各ウィンドウで実施

## 予測モデル設計指針

### アーキテクチャ

```
[データ収集]
  GMO FX API → USD/JPY 1H OHLC
        ↓
[特徴量生成]
  テクニカル指標（SMA, MACD, RSI, BB, ATR, ADX, StochRSI等）
  + 均衡乖離度（BB位置, RSI偏差, チャネル位置）  ← 資料3の知見
  + 時間帯特徴量（東京/ロンドン/NY セッション）
  + モメンタム（ROC, 上昇バー比率, ボラティリティ比率）
  + ラグ特徴量（リターンの1-12期ラグ）
        ↓
[ラベル生成]
  N足先の方向（上/下）
  閾値: X pips以上の動きのみラベル化（ノイズ除外）  ← 資料1,3の知見
        ↓
[モデル訓練]
  LightGBM（2値分類、予測確率出力）
  ウォークフォワード検証
  Optunaでハイパーパラメータ最適化
        ↓
[シグナルフィルタ]
  予測確率 > 閾値（例: 0.60-0.65）のときのみシグナル発行  ← 資料4の知見
  → 実効勝率を引き上げる
        ↓
[評価]
  ウォークフォワード各期間で:
  - 正答率（Accuracy）
  - 勝率（フィルタ後の実効勝率）
  - リスクリワード比
  - シャープレシオ
  - 最大ドローダウン
  - プロフィットファクター
```

### 最適化の優先順位（資料4に基づく）

1. **勝率の最大化**: モデル精度 + 予測確率フィルタ
2. **リスクリワード比の最大化**: 適切な損切り/利確ライン（ATRベース等）
3. **レバレッジの最小化**: 低レバレッジ運用を前提
4. **保有期間は気にしない**: 1時間足で十分

### 特徴量設計の方針（資料3に基づく）

「**定常的・有界な指標を重視**」する:

| 指標 | 範囲 | 性質 |
|------|------|------|
| RSI | 0-100 | 有界、平均回帰的 |
| ボリンジャーバンド位置 | 0-1 | 有界、平均回帰的 |
| ストキャスティクスRSI | 0-1 | 有界、平均回帰的 |
| チャネル位置 | 0-1 | 有界、平均回帰的 |
| ADX | 0-100 | 有界 |
| CCI | -∞〜+∞ | 平均回帰的 |
| ボラティリティレジーム (ATR/ATR_MA) | 0〜∞ | 平均回帰的 |
