## 概要

株価そのものではなく、共和分性を満たす株価ペアのスプレッドをLSTMで予測する手法を提案。AR(1)過程に従う人工データで事前学習したLSTMが、実データ直接学習よりも高精度な結果を達成した。

---

## 核心的アイデア

**「予測しやすいものを予測する」**

- 株価そのものは非定常（ランダムウォーク）であり、予測が極めて困難
- 共和分性を満たすペアのスプレッドは**定常過程**（平均回帰性を持つ）
- 定常過程は平均が時点に依らず一定 → 予測しやすい

---

## 提案手法の3ステップ

```
Step 1: 共和分性を満たす株価ペアからスプレッドを作成し、
        AR(1)過程を特徴づけるパラメータ（α, σ）を推定する

Step 2: 推定したパラメータを使いAR(1)過程に従う人工データを生成し、
        LSTMで学習する

Step 3: 学習したLSTMで実際のスプレッドの予測を行う
```

**ポイント**: 実際のデータを直接学習するのではなく、一旦パラメータを推定し、そのパラメータで生成した人工データをLSTMで学習させることが提案手法の特徴。

---

## 理論的背景

### 共和分性（Co-integration）

- 非定常な時系列データの線形結合が定常過程となる性質
- Engle and Granger (1987) によって提唱された
- ランダムウォークする2つの株価ペア {Xt, Yt} に対して、ある定数βが存在し:
  ```
  Yt = βXt + εt
  ```
  における誤差項 {εt} が定常となることが共和分性の条件
- スプレッド `St = Yt - βXt` は定常過程 → **平均回帰する**

### ペア・トレード戦略

- 価格変動が似通った銘柄ペアを見つける
- ペアの価格差（スプレッド）が均衡水準の周りを推移すると仮定
- スプレッドが均衡から乖離したとき、将来の平均回帰に賭けて:
  - 相対的に割高な方を売り
  - 割安な方を買う
- 標準的手法: ボリンジャーバンドで±2σ乖離時にポジション構築、均衡回帰時に解消

### AR(1)過程

```
St = α × St-1 + εt
（εt は正規分布 N(0, σ) に従う誤差項）
（St = Yt - βXt）
```

- **αが時間により変化する**ことが重要な観察
  - 従来研究ではαを一定として扱い → 予測精度の低下
  - TOPIX/日経平均ペアでの実測:
    - αの中央値: 0.7552
    - α ∈ {0.55, 0.65, 0.75, 0.85, 0.95} の5つの値を取ると推定
  - 1985年〜2006年の期間でαは大きく変動（図1参照）

### LSTMの役割

- **αの時間変化をモデル化する**ことがLSTMを使う意図
- 単純なAR過程ではパラメータの時間変化を扱えない
- LSTMは長期依存関係を学習できるため、パラメータの変動パターンを捉えられる

## 実装詳細

### LSTMネットワーク構成

| レイヤー | ユニット数 | 備考 |
|---------|---------|------|
| 入力層 | 1 | 全結合層に変更 |
| LSTM層 | 200 | |
| 出力層 | 1 | |

- **フレームワーク**: Chainer（オープンソース深層学習フレームワーク）
- **最適化**: Adam (Kingma and Ba, 2015)

### 学習パラメータ

- **AR(1)人工データ生成**:
  - 各αに対して時系列長20のサンプルを50個生成
  - 入力値は0〜1に基準化
  - バッチサイズ: 250
  - 学習回数: 600回（損失関数が十分に低下するまで）

- **実データ直接学習（比較対象 LSTM(raw)）**:
  - 1985年1月〜2006年12月のスプレッドから時系列長20のサンプルを250個生成
  - 同様に0〜1に基準化
  - バッチサイズ: 250、学習回数: 600回

## 実証分析

### 実験設定

| 項目 | 値 |
|------|------|
| 対象ペア | TOPIX指数 × 日経平均株価 |
| データソース | Bloomberg |
| 学習期間 | 1985年1月〜2006年12月末 |
| 検証期間 | 2007年1月〜2017年7月末 |
| 予測対象 | 20営業日後のスプレッド |
| β推定タイミング | 年始最初の営業日に推定、毎年更新 |
| σの推定期間 | 1985年1月〜2006年12月末 |

### 収益率計算

```
予測が上昇の場合: (St+20 - St) / (Y + abs(βX))
予測が下落の場合: -(St+20 - St) / (Y + abs(βX))
上記を20で割った値を日次収益率とした
```

### 結果: 正答率・収益率

| 手法 | 正答率 | 収益率 |
|------|--------|--------|
| AR(1) | 56.01% | 13.45% |
| LSTM(raw) — 実データ直接学習 | 56.59% | 14.06% |
| **LSTM(AR) — 提案手法** | **58.33%** | **17.26%** |

### 結果: F値の詳細

| 手法 | クラス | Precision | Recall | F1-score |
|------|--------|-----------|--------|----------|
| AR(1) | 下落(0) | 50% | 63% | 56% |
| AR(1) | 上昇(1) | 63% | 51% | 56% |
| AR(1) | 平均 | 57% | 56% | 56% |
| LSTM(raw) | 下落(0) | 52% | 25% | 34% |
| LSTM(raw) | 上昇(1) | 58% | 81% | 68% |
| LSTM(raw) | 平均 | 55% | 57% | 53% |
| **LSTM(AR)** | **下落(0)** | **53%** | **46%** | **50%** |
| **LSTM(AR)** | **上昇(1)** | **62%** | **68%** | **65%** |
| **LSTM(AR)** | **平均** | **58%** | **58%** | **58%** |

### 結果の解釈

1. **LSTM(raw)はAR(1)の結果を再現**: LSTMがスプレッドの時系列を学習する能力を持つことを確認
2. **LSTM(AR)はAR(1)とLSTM(raw)の両方を上回る**: 正答率・収益率ともに最良
3. **収益率推移でLSTM(AR)がLSTM(raw)を安定して上回る**: 2007-2017年の期間を通じて一貫
4. **人工データ学習 > 実データ直接学習**: ノイズを除去し、本質的なダイナミクスを学習

---

## 重要な考察

### なぜ人工データ事前学習が有効か

- 実データにはノイズが多い
- AR(1)過程という**構造的仮定**を通じてノイズを除去
- 本質的なダイナミクス（平均回帰の速度とその変化）のみを学習
- AR(1)を用いた共和分時系列分析の妥当性を示唆

### パラメータの時間変化の重要性

- αを固定すると予測精度が低下する
- LSTMでαの変動を捉えることが精度向上の鍵
- 市場環境（レジーム）が時間とともに変化することの表れ

### 適用範囲

- αは他の株価ペアにも適用できる一般的な値と考えられる
- σをデータから推定するだけで他の対象にも応用可能
- 分散を計測するのに十分な短期間のデータがあれば適用可能

---

## 本プロジェクトへの示唆

| 知見 | 適用 |
|------|------|
| 「予測しやすいものを予測する」 | 価格の方向ではなく、テクニカル指標の均衡からの乖離度を特徴量に |
| 定常性のある系列は予測精度が高い | RSI, BB位置, ストキャスティクス等は有界・平均回帰的 |
| 人工データ事前学習の有効性 | データ拡張やノイズ除去の考え方として参考 |
| パラメータの時間変化 | 市場レジーム（トレンド/レンジ）の変化を前提とした設計 |

### USD/JPY単体への応用

共和分ペアトレードはUSD/JPY単体には直接適用できないが、**考え方は応用可能**:

- テクニカル指標の多くは**有界**であり**平均回帰的**:
  - RSI: 0-100
  - ボリンジャーバンド位置: 0-1
  - ストキャスティクスRSI: 0-1
  - チャネル位置: 0-1
- 「指標が極端な値から均衡に戻る方向」は「価格の方向そのもの」より予測しやすい
- ボラティリティレジーム（ATR/ATR移動平均）も定常的な振る舞いをする
