# リアル取引ロードマップ

## 概要

v2モデル（模擬FW勝率55.7%、PF1.34）の実績を段階的に確認しながら、リアル取引に移行する計画。

## 現在のモデル性能（模擬フォワード）

| 指標 | 値 | 世界水準との比較 |
|------|-----|----------------|
| 勝率 | 55.7% | ヘッジファンドでも60%未満。論文水準に近い |
| PF | 1.34 | 1.5以上が「強い」。まずまずの水準 |
| 取引頻度 | 3.9回/日 | 十分な頻度 |
| スプレッド耐性 | 2.0pipsでもプラス | 堅牢 |

---

## 段階的投資計画

### Phase 2: 仮想取引（現在）

| 項目 | 内容 |
|------|------|
| 期間 | 1-2週間（〜2026-04-01頃） |
| 資金 | 0円（仮想取引） |
| 目的 | v2モデルのリアルタイム実績確認 |
| 判定基準 | シグナルが出ているか、決済が正常か |
| 次へ進む条件 | 取引10回以上、勝率50%以上 |

### Phase 3-A: 最小資金テスト

| 項目 | 内容 |
|------|------|
| 期間 | 1ヶ月 |
| 資金 | **1万円** |
| レバレッジ | 5倍 |
| 目的 | リアル取引の動作確認（API連携、注文執行、決済） |
| 月間期待利益 | +1,000〜2,000円（模擬FWの1/3見積もり） |
| 最悪月損失 | -2,000円 |
| 次へ進む条件 | PF > 1.0（利益 > 損失） |
| 撤退条件 | PF < 0.8 または DD > 30% |

### Phase 3-B: 本格運用

| 項目 | 内容 |
|------|------|
| 期間 | 2-3ヶ月 |
| 資金 | **5万円** |
| レバレッジ | 5倍 |
| 目的 | 安定した利益確認 |
| 月間期待利益 | +5,000〜10,000円 |
| 最悪月損失 | -10,000円 |
| 次へ進む条件 | 3ヶ月連続でPF > 1.0 |
| 撤退条件 | 2ヶ月連続マイナス |

### Phase 3-C: 増額運用

| 項目 | 内容 |
|------|------|
| 期間 | それ以降 |
| 資金 | **利益 + 追加資金（10万円〜）** |
| レバレッジ | 5倍 |
| 目的 | 本格的な利益獲得 |
| 月間期待利益 | +10,000〜20,000円（10万円運用時） |
| 最悪月損失 | -20,000円 |

## 資金別シミュレーション（レバ5倍/閾値0.54、複利運用）

| 開始資金 | 8ヶ月後（模擬FW上限） | 現実的な期待値（1/3） | 最大DD |
|---------|---------------------|-------------------|--------|
| 1万円 | 43,389円 | 約20,000円 | 50% |
| 5万円 | 216,945円 | 約100,000円 | 50% |
| 10万円 | 433,890円 | 約200,000円 | 50% |

※模擬FWの数字は上限値。実運用では1/3〜1/5程度が現実的。

---

## 重要ルール

1. **失ってもいい金額だけ入れる** — 借金は絶対にしない
2. **段階的に増やす** — いきなり大金を入れない
3. **撤退条件を守る** — PF < 0.8で即停止、感情で続けない
4. **月次で評価** — 毎月末にPF・勝率・DDを確認
5. **モデルの再訓練** — 月1回、SW3年窓をスライドして更新

## Phase 3-Aに必要な実装

| タスク | 内容 | 優先度 |
|--------|------|--------|
| GMO FXリアル口座開設 | Private APIアクセスの取得 | 必須 |

| 残高管理 | マイナス防止、最低証拠金チェック | 必須 |
| 複利対応 | 残高に応じたロット計算 | 推奨 |
| 日次損失リミット | 日次-3%で当日停止 | 推奨 |
| 週末持ち越し禁止 | 金曜NY引け前に全決済 | 推奨 |
