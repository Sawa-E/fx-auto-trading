## v2モデル構成

| 項目 | v1（旧） | **v2（新）** |
|------|---------|------------|
| 訓練窓 | 拡大窓10年 | **スライディングウィンドウ3年**（資料3） |
| キャリブレーション | なし | **sigmoid**（資料5） |
| 特徴量 | 15個（RF重要度選択） | **23個（定常重視）**（資料3） |
| 特徴量の有界率 | 不明 | **70%** |
| 閾値 | 0.53 | **0.54** |
| パラメータ | Optuna 50trial | **固定パラメータ**（資料5: チューニングやりすぎ防止） |
| ホライゾン | N=8（8時間先） | N=8（同じ） |
| レジームフィルタ | ADX > 25 | ADX > 25（同じ） |
| 損切り/利確 | ATR×1.5 / ATR×2.0 | 同じ |

### 改善の論文根拠

| 改善 | 内容 | 論文根拠 |
|------|------|---------|
| A: スライディングウィンドウ | 拡大窓10年→直近3年 | 資料3（SIG-FIN）: 「パラメータαは時間変化する。固定すると予測精度が低下」 |
| B: 確率キャリブレーション | Platt scaling (sigmoid) | 資料5（MUFG）: 「マーケットデータは再現性が圧倒的に低い」→ 保守的な確率出力を補正 |
| D: 定常特徴量 | 有界・平均回帰指標を重視（23個） | 資料3: 「定常的な系列は予測しやすい」、非定常ラグ特徴量を除外 |

### 閾値選択の判断

| 戦略 | 閾値 | 特徴 |
|------|------|------|
| 累積利益最大・安定 | 0.51 | +24,512円、6/8プラス、12回/日 |

## 改善過程のまとめ

### 試した改善案と結果

| 改善 | 内容 | 模擬FW合計損益 | 判定 |
|------|------|-------------|------|
| 現行v1（拡大窓/なし/0.53） | ベースライン | -2,871円 | × |
| B: キャリブレーションのみ | Platt scaling追加 | -2,767円 | △ 微改善 |
| **A: SW3年のみ** | **拡大窓→直近3年** | **+16,545円** | **◎ 最大の改善** |
| A+B: SW3年+キャリブ | SW+Platt | +17,112円 | ◎ |
| **A+B+D: SW+キャリブ+定常特徴量** | **全改善** | **+25,329円** | **◎ 最高** |
| A+B+E: SW+キャリブ+XGBアンサンブル | +XGBoost | +15,266円 | △ 逆効果 |
| A+B+D+E: 全部入り | 全改善+XGB | +17,393円 | ○ D+Eは相殺 |

### 効果の大きさ順

1. **A（スライディングウィンドウ）**: -2,871円→+16,545円。**最大の効果**。古いデータが足を引っ張っていた
2. **D（定常特徴量）**: +17,112円→+25,329円。有界指標の重視が効いた
3. **B（キャリブレーション）**: 単体では微改善だが、Aと組み合わせて効果を発揮
4. **E（XGBoostアンサンブル）**: 逆効果。パラメータ未最適化のXGBoostが足を引っ張る

## 3万円運用シミュレーション

| 項目 | 値 |
|------|-----|
| 開始資金 | 30,000円 |
| レバレッジ | 5倍 |
| 月間平均損益 | **+5,882円** |
| 月間平均利回り | **+19.6%**（模擬FW、実運用では5-10%が現実的） |
| 最悪月損失 | -1,976円（-6.6%） |

### 注意事項

- 模擬FWの月利19.6%は**上限値**。実運用では大幅に下がる見込み
- 最大DD 57.7%は2024/8の暴落→回復期間。実資金なら精神的に耐えにくい
- 8期間中2期間はマイナス（BOJ介入、円高）。買い専用モデルの限界
